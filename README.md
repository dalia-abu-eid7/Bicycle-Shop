# Velocity Cycles – E-commerce Bicycle Store

## Overview
Velocity Cycles is a high-performance, responsive e-commerce front-end project designed for modern bicycle retailers. This project focuses on a "Mobile-First" approach, ensuring that customers have a seamless browsing and shopping experience whether they are on a smartphone, tablet, or desktop.

## Key Features

Dynamic Hero Section: Engaging visuals with clear Call-to-Action (CTA) buttons to highlight new arrivals.

Product Catalog: A well-structured grid display for various bicycle categories (Mountain, Road, Electric) with hover effects.

Service & Maintenance Section: Dedicated area to showcase professional repair services and tuning packages.

Interactive UI Elements: Functional navigation bar, product filtering visuals, and smooth scrolling animations using JavaScript.

Shopping Cart UI: A clean and intuitive cart layout to preview selected items.

Newsletter & Contact: Integrated forms for customer retention and direct inquiries.

## Technical Stack

HTML5 & CSS3: For semantic structure and custom modern styling.

Bootstrap 5: Leveraged for the grid system and pre-built components to ensure 100% responsiveness.

JavaScript (ES6+): Used for DOM manipulation, interactive menus, and dynamic content updates.

Google Fonts & FontAwesome: For sleek typography and high-quality iconography.
